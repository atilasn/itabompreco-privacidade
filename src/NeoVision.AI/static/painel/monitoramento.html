Tipo (ícone no mapa)

Com URL preenchida, a NeoVision faz um pedido HTTP (GET) a este endereço — porta e caminho incluídos — em vez de ping ICMP. Vazio = só ICMP ao IP acima. Vale para qualquer tipo de equipamento que escolher no mapa.
Painel web DVR / NVR (opcional)

Login automático (Basic Auth): no campo URL use só o endereço completo, por exemplo http://admin:sua_senha@10.100.103.8:8008/ — não precisa preencher usuário e senha abaixo. Se preferir, use URL sem login e preencha usuário e senha; o efeito é o mesmo. Muitos DVR usam tela de login em HTML: nesse caso não dá para entrar sozinho; só funciona quando o equipamento aceita Basic Auth na URL. Na app NeoVision, «Abrir DVR» tenta primeiro uma janela interna; o painel embutido (tela inteira) pode ficar branco se o gravador bloquear.

Ao editar, deixe em branco para manter a senha guardada. Marque «Limpar» para apagar só a senha.
